Case Study 1: Tabular Classifier

Feature Distributions

Analysis: Strong right-skewness in 'Orbital period', 'Transit duration', and 'Planet radius'.
Conclusion: Logarithmic transformation required to stabilize variance.

PCA Projection

Analysis: Classes overlap heavily in PCA space.
Conclusion: Non-linear classifiers necessary for separation.

t-SNE Projection

Analysis: Class 1 is scattered and ambiguous.
Conclusion: Predictive difficulty expected for Class 1.

Model Comparison

Analysis: All models within 0.72–0.74 accuracy.
Conclusion: RandomForest/XGBoost chosen for efficiency.

Learning Curve

Analysis: Large gap between training and validation.
Conclusion: High variance — strong overfitting detected.

Confusion Matrix

Analysis: Class 1 has high misclassification.
Conclusion: Major improvement required for Class 1 accuracy.

ROC Curves

Analysis: Class 1 curve is close to random.
Conclusion: Model fails to separate Class 1 well.

Feature Importance

Analysis: signal_to_noise dominates feature set.
Conclusion: Detection quality drives predictions.

SHAP Explainability

Analysis: SHAP confirms importance of signal_to_noise and transit_duration.
Conclusion: Local + global interpretability aligned.
